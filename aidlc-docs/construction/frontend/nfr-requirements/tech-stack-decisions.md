# Frontend - 기술 스택 결정

## 핵심 스택
| 항목 | 결정 | 버전 |
|------|------|------|
| Language | TypeScript | 5.x |
| Framework | React | 18.x |
| Build Tool | Vite | 5.x |
| 상태 관리 | Zustand | 4.x |
| 라우팅 | React Router | 6.x |
| HTTP Client | Axios | 1.x |

| 라이브러리 | 용도 |
|-----------|------|
| zustand | 상태 관리 + persist middleware |
| react-router-dom | SPA 라우팅 |
| axios | HTTP API 호출 |
| eslint | 코드 린팅 |
| prettier | 코드 포맷팅 |
